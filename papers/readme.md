# Video Frame Selection Papers

## Overview
This repository contains a curated list of papers on video frame selection for long-form video understanding, with a focus on surveillance videos and multimodal large language models (MLLMs).

---

### üîÑ Reinforcement Learning Methods

#### A.I.R (Adaptive, Iterative, and Reasoning-based Frame Selection)
- **Title**: Enabling Adaptive, Iterative, and Reasoning-based Frame Selection for Video Question Answering
- **Link**: [OpenReview](https://openreview.net/forum?id=SZVpOKw0YD)
- **Status**: Submitted to ICLR2026 (not accepted yet)
- **Description**: Effectively applying Vision-Language Models (VLMs) to VideoQA hinges on selecting a concise yet comprehensive set of frames

### üîß Training-Free Methods

#### FOCUS
- **Title**: Efficient Keyframe Selection for Long Video Understanding
- **Link**: [OpenReview](https://openreview.net/forum?id=1OQKqLFcbB)
- **Status**: Submitted to ICLR2026 (not accepted yet)
- **Description**: MLLMs represent images and video frames as visual tokens. Scaling from single images to hour-long videos inflates the token budget far beyond practical limits
